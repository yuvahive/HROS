💾 DATA STORAGE (No Backend Needed!)

Everything stored locally in browser via localStorage:

localStorage['hros_users']
├─ User database (array of users with details)

localStorage['hros_current_user']
├─ Currently logged-in user session (or null)

localStorage['hros_idp_config']
├─ OAuth/SAML identity provider configuration

localStorage['hros_hiring_board']
localStorage['hros_daily_work_board']
└─ All existing HROS board data (unchanged)

✅ All data persists across browser refreshes
✅ No server required
✅ GDPR compliant (stays in browser)
✅ Fast access & offline support
